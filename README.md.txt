# HTML Quick Styler

HTML Quick Styler is a simple AI-powered web application that helps users improve and style HTML code quickly. The application takes HTML input from the user and generates styled or improved HTML output using an AI model.

## Features

- AI-assisted HTML styling
- Simple web interface using Gradio
- Quick HTML formatting and improvements
- Easy to run locally

## Technologies Used

- Python
- Gradio
- Transformers (Hugging Face)
- PyTorch
